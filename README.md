# Tidepool Go Client

Go client library for the Tidepool Query and Ingest services.

## Status

This repository currently contains the public design document for the client. The implementation is not yet included.

## Documentation

- `tidepool-go-client-design.md` — API contract and usage examples for the planned client.

## Planned Package Layout

```
tidepool/
├── client.go       // Main client implementation
├── types.go        // Data types and enums
├── errors.go       // Error types
└── options.go      // Functional options
```

## Installation

Once the client is released:

```bash
go get github.com/your-org/tidepool-go
```
